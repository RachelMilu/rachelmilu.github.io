css 垂直居中

1.单行文本垂直居中

vertical-align属性 它只对(X)HTML元素中拥有valign特性的元素才生效，例如表格元素中的td、th、caption等, 而像div、span这样的元素是没有valign特性的，因此使用vertical-align对它们不起作用。

使用line-height,一般会用overflow:hidden来防止内容超出容器或者产生自动换行

div{ height:25px; line-height:25px; overflow:hidden; }

2.多行文本垂直居中

通过display属性使div标签模拟table标签，即外部divdisplay为table，内部div的display为table-cell，vertical-align为middle

缺陷：内部div不能同时使用float属性，否则居中会失效。
